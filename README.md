genregen
========

A tiny music genre generator.

## Usage

### Installation

``` bash
$ gem install genregen
```

### Ruby

```ruby
GenreGen.generate
```

### Shell

```bash
$ genregen
```

## Contributing

Improvements to the generator and dataset are welcomed and appreciated.

When adding to [lib/data.yml](data.yml), please preserve alphabetical order.
